energy\_toolbox.timeseries.profiles package
===========================================

.. automodule:: energy_analysis_toolbox.timeseries.profiles

The `profiles` subpackage is a direct migration of the previously existing
`load_profiles` library. It contains functions to compute profiles from
historical variations of a quantity represented as flat timeseries data.
